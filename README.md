# 王者荣耀自动点击器 - Android版 v2.0

## 🎮 项目简介

这是一个基于 **Android 无障碍服务** 实现的王者荣耀自动化点击器。

### ✨ v2.0 新特性

- 🔧 **Android 原生实现** - 使用无障碍服务，无需 ADB
- 📱 **自动适配屏幕** - 自动缩放坐标，支持各种分辨率
- 🎯 **11步完整流程** - 从登录到返回大厅全自动化
- ⚙️ **可视化配置** - 图形界面配置坐标参数
- 📊 **实时日志** - 运行状态一目了然

## 📋 功能列表

### 11步自动化流程

| 步骤 | 名称 | 说明 |
|------|------|------|
| 1 | 登录 | 点击"开始游戏"登录 |
| 2 | 关闭弹窗 | 关闭活动弹窗 |
| 3 | 游戏大厅 | 点击"对战" |
| 4 | 匹配 | 点击"王者峡谷" |
| 5 | AI模式 | 选择"人机" |
| 6 | 开始游戏 | 点击"开始" |
| 7 | 准备游戏 | 点击"准备" |
| 8 | 准备进入 | 确认进入游戏 |
| 9 | 游戏结束 | 等待并确认结束 |
| 10 | 结算英雄 | 查看结算界面 |
| 11 | 返回房间 | 返回游戏大厅 |

## 📱 系统要求

- **Android 版本**: 7.0 (API 24) 及以上
- **权限要求**:
  - 无障碍服务 (必须)
  - 存储权限 (必须)
  - 悬浮窗权限 (推荐)

## 🚀 使用方法

### 安装

1. 下载最新 APK 文件
2. 安装到 Android 设备
3. 打开应用

### 首次使用

1. 切换到 **"权限"** 标签页
2. 点击 **"打开无障碍设置"**
3. 找到 **"王者荣耀自动点击器"**
4. 开启无障碍服务权限
5. 返回应用

### 开始使用

1. 切换到 **"主控制"** 标签页
2. 确认无障碍服务状态为 ✅ 已启用
3. 选择脚本（默认：智能自动点击器）
4. 点击 **"开始运行"**

### 配置参数

1. 切换到 **"配置"** 标签页
2. 调整各步骤坐标（可选）
3. 设置等待时间、匹配阈值等
4. 点击 **"保存配置"**

## 🛠️ 开发说明

### 本地开发

```bash
# 安装依赖
pip install kivy pyjnius opencv-python numpy

# 运行测试
python main.py
```

### 构建 APK

本项目使用 GitHub Actions 自动构建。

1. Fork 本仓库
2. 修改代码
3. Push 到 main 分支
4. 等待 Actions 构建完成
5. 在 Releases 下载 APK

### 手动构建

```bash
# 安装 buildozer
pip install buildozer

# 初始化
buildozer init

# 构建 APK
buildozer android debug
```
