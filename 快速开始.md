# 快速开始指南

## 🎯 目标

在30分钟内完成APK构建，获得修复了所有问题的版本。

## ⚡ 最快方式：GitHub Actions

### 步骤1：准备GitHub仓库（5分钟）

1. 访问 https://github.com/new
2. 创建新仓库：
   - 名称：`wangzhe-autoclicker`
   - 类型：Public 或 Private
   - **不要**勾选任何初始化选项

### 步骤2：上传代码（5分钟）

打开PowerShell，进入项目目录：

```powershell
cd C:\Users\Administrator\.openclaw\workspace\wangzhe_autoclicker_v3

# 初始化Git
git init
git add .
git commit -m "v3.0: 完整重构版本 - 修复所有问题"

# 设置分支
git branch -M main

# 添加远程仓库（替换YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/wangzhe-autoclicker.git

# 推送
git push -u origin main
```

### 步骤3：等待构建（20分钟）

1. 访问你的仓库：`https://github.com/YOUR_USERNAME/wangzhe-autoclicker`
2. 点击 **Actions** 标签
3. 查看构建进度（约20分钟）
4. 等待绿色勾勾 ✅

### 步骤4：下载APK

1. 点击完成的workflow
2. 在 **Artifacts** 部分下载 `wangzhe-autoclicker-v3-apk`
3. 解压得到APK文件

### 步骤5：安装测试

```powershell
# 连接设备
adb devices

# 安装APK
adb install wangzheautoclicker_v3.0_*.apk

# 启动应用
adb shell am start -n org.wangzhe.wangzheautoclicker/org.kivy.android.PythonActivity
```

## 🐛 遇到问题？

### 构建失败
- 查看Actions日志
- 通常是依赖问题，重新触发构建即可

### Git推送失败
```powershell
# 检查远程仓库
git remote -v

# 重新设置
git remote set-url origin https://github.com/YOUR_USERNAME/wangzhe-autoclicker.git

# 再次推送
git push -u origin main
```

### APK安装失败
```powershell
# 检查设备架构
adb shell getprop ro.product.cpu.abi

# 卸载旧版本
adb uninstall org.wangzhe.wangzheautoclicker

# 重新安装
adb install wangzheautoclicker_v3.0_*.apk
```
