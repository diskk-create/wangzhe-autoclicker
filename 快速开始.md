# 🚀 快速开始 - 3步获取APK

## 第1步：创建GitHub仓库（2分钟）

1. 访问 https://github.com/new
2. 填写信息：
   - Repository name: `wangzhe-autoclicker`
   - Description: `王者荣耀自动点击器 Android版`
   - 选择 **Public**（公开）
   - **不要勾选** "Add a README file"
   - **不要勾选** "Add .gitignore"
3. 点击 **Create repository**

## 第2步：上传项目（3分钟）

### 方法A：使用GitHub网页（最简单）

1. 在刚创建的仓库页面，点击 **"uploading an existing file"**
2. 打开文件夹：`C:\Users\Administrator\.openclaw\workspace\王者荣耀自动点击器_APK`
3. **拖拽所有文件**到网页中
4. 点击 **Commit changes**

### 方法B：使用Git命令（推荐）

打开PowerShell，运行：

```bash
cd "C:\Users\Administrator\.openclaw\workspace\王者荣耀自动点击器_APK"

git init
git add .
git commit -m "初始提交：王者荣耀自动点击器"
git branch -M main
git remote add origin https://github.com/你的用户名/wangzhe-autoclicker.git
git push -u origin main
```

## 第3步：下载APK（等待自动构建）

1. 上传完成后，等待30秒
2. 点击仓库页面的 **"Actions"** 标签
3. 查看构建进度（通常需要20-40分钟）
4. 构建完成后，点击构建记录
5. 在 **"Artifacts"** 区域下载 `wangzhe-autoclicker-apk.zip`
6. 解压得到APK文件

## ✅ 完成！

- APK文件：`王者荣耀自动点击器-1.0.0-arm64-v8a-debug.apk`
- 安装到手机即可使用！

## 🎯 如果遇到问题

### 问题1：没有GitHub账号
- 访问 https://github.com/signup 免费注册

### 问题2：构建失败
- 检查Actions页面的错误日志
- 通常是因为文件缺失，确保所有文件都上传了

### 问题3：找不到Artifacts
- 确保构建已完成（绿色对勾）
- Artifacts在构建详情页面的底部
