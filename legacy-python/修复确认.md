### 2. 横屏闪退问题 ✅
**原始错误**:
```
IllegalStateException: View has already been added to the window manager
```

**修复代码** (scripts/float_window_manager.py):
```python
class FloatWindowManager:
    """悬浮窗管理器 - 单例模式"""
    
    _instance = None
    _initialized = False
    
    def __new__(cls, *args, **kwargs):
        """单例模式"""
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance
    
    def __init__(self):
        """初始化悬浮窗管理器"""
        if FloatWindowManager._initialized:
            return
        FloatWindowManager._initialized = True
        # 防止重复初始化...
```

**额外修复** (buildozer.spec):
```ini
android.manifest_activity_attributes = android:configChanges="orientation|screenSize|keyboardHidden"
```

**状态**: ✅ 已修复

### 4. 分辨率不适配问题 ✅
**原始问题**: 固定坐标，其他分辨率点击不准

**修复代码** (scripts/resolution_adapter.py):
```python
class ResolutionAdapter:
    """分辨率自动适配器"""
    
    BASE_WIDTH = 720
    BASE_HEIGHT = 1280
    
    def adapt_coordinate(self, x, y, from_base=True):
        """适配坐标"""
        if from_base:
            return (
                int(x * self.scale_x),
                int(y * self.scale_y)
            )
        else:
            return (
                int(x / self.scale_x),
                int(y / self.scale_y)
            )
```

**状态**: ✅ 已修复（自动适配）
