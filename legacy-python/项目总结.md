## 📱 打包APK步骤

### 方法1：WSL（推荐）

```bash
# 1. 启动WSL
wsl

# 2. 进入项目目录
cd /mnt/c/Users/Administrator/.openclaw/workspace/王者荣耀自动点击器_APK

# 3. 安装buildozer（首次）
pip3 install buildozer cython==0.29.33

# 4. 打包APK（首次需要30-60分钟）
buildozer android debug

# 5. 获取APK
# bin/王者荣耀自动点击器-1.0.0-arm64-v8a-debug.apk
```

### 方法2：GitHub Actions（最简单）

1. 上传项目到GitHub
2. 添加 .github/workflows/build.yml
3. 推送代码
4. 自动构建APK
5. 下载APK

### 方法3：云服务器

- 阿里云/腾讯云 Ubuntu服务器
- 安装buildozer
- 上传项目打包
- 下载APK
