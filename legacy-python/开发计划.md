**代码**:
```python
# 添加到 main.py
def adapt_coordinate(self, x, y):
    """适配坐标"""
    base_width = 1280
    base_height = 720
    scale_x = self.screen_width / base_width
    scale_y = self.screen_height / base_height
    return int(x * scale_x), int(y * scale_y)
```

## 💡 开发原则

1. **渐进式开发**
   - 先确保基础功能工作
   - 再逐步添加高级功能
   - 每次只改一个功能

2. **验证优先**
   - 每次修改后立即测试
   - 不积压问题
   - 确保每一步都可用

3. **代码简洁**
   - 保持代码简单
   - 避免过度设计
   - 先让它工作，再让它完美

4. **文档完整**
   - 记录每个阶段
   - 保存测试结果
   - 方便回溯问题
