# 🎯 图文教程 - 从零开始获取APK

## 📋 准备工作

你需要：
- ✅ 一个GitHub账号（免费注册：https://github.com/signup）
- ✅ 5分钟时间上传文件
- ✅ 等待20-40分钟自动构建

---

## 第1步：注册GitHub账号（2分钟）

### 1.1 访问注册页面
打开浏览器，访问：https://github.com/signup

### 1.2 填写信息
- **Email**: 你的邮箱
- **Password**: 设置密码
- **Username**: 用户名（英文）
- 点击 **Create account**

### 1.3 验证邮箱
- 登录你的邮箱
- 找到GitHub的验证邮件
- 点击验证链接

**✅ GitHub账号创建完成！**

---

## 第2步：创建GitHub仓库（1分钟）

### 2.1 访问创建页面
登录后，访问：https://github.com/new

或点击右上角 **+** → **New repository**

### 2.2 填写仓库信息

```
Repository name: wangzhe-autoclicker
Description: 王者荣耀自动点击器Android版
Visibility: ✅ Public（选择公开）

⚠️ 重要：不要勾选以下选项！
[ ] Add a README file
[ ] Add .gitignore
[ ] Choose a license
```

### 2.3 点击创建
点击绿色按钮 **Create repository**

**✅ 仓库创建完成！**

---

## 第3步：上传项目文件（3分钟）

### 方法A：网页上传（最简单）⭐⭐⭐⭐⭐

#### 3.1 打开上传页面
在刚创建的仓库页面，点击 **"uploading an existing file"**

#### 3.2 准备文件
打开文件夹：
```
C:\Users\Administrator\.openclaw\workspace\王者荣耀自动点击器_APK
```

#### 3.3 拖拽上传
将以下**所有文件和文件夹**拖到网页中：

**必须上传的文件：**
```
✅ main.py
✅ buildozer.spec
✅ requirements.txt
✅ README.md
✅ 快速开始.md
✅ .gitignore
✅ .github/workflows/build.yml
✅ scripts/ 文件夹（包含所有.py文件）
✅ templates/ 文件夹（包含所有.png文件）
✅ configs/ 文件夹（包含config.json）
```

**快捷方式：**
- 按 `Ctrl+A` 全选所有文件
- 拖拽到网页中

#### 3.4 提交更改
- 在页面底部，填写：
  - Commit message: `初始提交：王者荣耀自动点击器`
- 点击 **Commit changes** 绿色按钮

**✅ 文件上传完成！**

---

## 第4步：等待自动构建（20-40分钟）

### 4.1 查看构建状态
上传完成后：
1. 点击仓库顶部的 **"Actions"** 标签
2. 会看到一个正在运行的任务：**Build Android APK**
3. 点击进入查看详情

### 4.2 查看构建进度
你会看到：
- ✅ Checkout code
- ✅ Set up Python 3.9
- ✅ Install system dependencies
- ✅ Install Python dependencies
- ⏳ Build APK with Buildozer（这里需要20-30分钟）
- ✅ Upload APK artifact

### 4.3 构建完成标志
当所有步骤都显示绿色对勾 ✅，表示构建成功！

**✅ APK构建完成！**

---

## 第5步：下载APK（1分钟）

### 5.1 找到Artifacts
1. 在构建详情页面，滚动到最底部
2. 找到 **"Artifacts"** 区域
3. 点击 **"wangzhe-autoclicker-apk"**

### 5.2 解压文件
下载的是一个ZIP文件，解压后得到：
```
王者荣耀自动点击器-1.0.0-arm64-v8a-debug.apk
```

**✅ APK下载完成！**

---

## 第6步：安装APK（1分钟）

### 6.1 传输APK到手机
选择一种方式：
- 微信/QQ发送文件到手机
- 数据线传输
- 网盘下载

### 6.2 安装APK
1. 在手机上点击APK文件
2. 如果提示"禁止安装"，去设置允许
3. 点击 **"安装"**
4. 等待安装完成

### 6.3 授予权限
首次打开应用时：
- ✅ 允许存储权限
- ✅ 允许无障碍服务
- ✅ 允许其他必要权限

**✅ 安装完成！开始使用！**

---

## 🎮 使用方法

### 打开应用
1. 在手机桌面找到"王者荣耀自动点击器"
2. 点击打开

1. 点击 **"配置编辑"** 标签

3. 点击 **"保存配置"**

## ❓ 常见问题

### Q1: 构建失败怎么办？
**A:** 查看Actions页面的错误日志，通常是：
- 文件缺失（确保所有文件都上传了）
- 语法错误（检查代码）

### Q2: 找不到Artifacts？
**A:** 确保：
- 构建已完成（绿色对勾）
- 滚动到页面最底部

### Q3: APK安装失败？
**A:** 确保：
- 手机Android版本 ≥ 5.0
- 允许安装未知来源应用

### Q4: 应用闪退？
**A:** 检查：
- 是否授予了所有权限
- 查看logcat日志

### Q5: 如何更新APK？
**A:**
- 修改代码后，推送到GitHub
- 自动重新构建
- 下载新的APK
